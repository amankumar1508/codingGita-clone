# 🎓 codingGita Student Portal

![React](https://img.shields.io/badge/React-19-61DAFB?style=for-the-badge&logo=react)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-4.0-06B6D4?style=for-the-badge&logo=tailwindcss)
![Vite](https://img.shields.io/badge/Vite-7.0-646CFF?style=for-the-badge&logo=vite)

A high-performance, developer-centric Student Management System designed for the **codingGita** ecosystem. This platform streamlines campus life by providing students with a centralized hub for attendance, schedules, and performance tracking.

---

## ✨ Key Features

- **🚀 Interactive Dashboard**: A bird's-eye view of your academic progress, upcoming tasks, and recent announcements.
- **📅 Smart Attendance**: Track your daily and semester-wise attendance with intuitive progress visualizations.
- **📆 Integrated Calendar**: Never miss a session with a dynamic schedule and event management system.
- **💬 Real-time Chat**: Seamless communication with peers and mentors built directly into the portal.
- **📝 Feedback System**: Comprehensive feedback modules for general campus experience and weekly subject-specific reviews.
- **🏥 Leave Management**: Apply for leaves and track approval status with ease.
- **👤 Personalized Profiles**: Manage your student identity and academic records in one secure place.

---

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://react.dev/) (latest features, high performance)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/) (next-gen utility-first CSS)
- **Routing**: [React Router 7](https://reactrouter.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
Make sure you have **Node.js** (v18+) installed.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/codingGita-clone.git
   ```
2. Navigate to the project directory:
   ```bash
   cd codingGita-clone/my-project
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
Start the development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to see the result.
